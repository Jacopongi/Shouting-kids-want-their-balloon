SHOUTING KIDS WANT THEIR BALLOON
Intelligent Distributed Systems Project 

Authors: Jacopo Endrizzi & Thade Pfluger 
Course: Intelligent Distributed Systems
Professor: Daniele Fontanelli
Course Year: 2023-2024

FOLDER DESCRIPTION:
- videos			  -> videos of all cases with actual positions
- figures report  		  -> .jpeg storage of figures shown in report
- warehouse 			  -> multiple runs matrices and excel files storage
- museum archive 		  -> old figures/videos

FILE DESCRIPTION:
- CaseDistinction 	          -> choose btw different cases
- ChooseSensorNumber 	          -> consensus algorithm for sensors distribution
- distributeKidBalloon 	          -> initial positions kids and balloons
- distributeKidBalloonForFlocking -> initial positions kids and balloons (for Flocking.m)
- distributeSensorsOnPerimeter 	  -> deploy sensors on walls
- distributeTriangle 		  -> auxiliary function for triangular initial positions
- EstimatePosition 		  -> consensus algorithm for estimate position
- EstimatePositioNoCons           -> single estimate position (no consensus algorithm)
- Flocking 			  -> flocking behaviour implementation
- FlockingRuns 		          -> multiple flocking runs
- intersections 		  -> auxiliary function 
- IDS_Report_ENDRIZZI_PFLUGER	  -> report 
- isInside 			  -> check if positions are inside boundaries
- MakeBalloon 		          -> struct creation 
- MakeKid 			  -> struct creation 
- MultipleMain 			  -> wrapper file to run runrun.m several times 
- MultipleRun 			  -> wrapper file to run Flocking.m several times 
- PlotMetrics 			  -> plot some metrics 
- RendezvousGame 		  -> pick a random position and the flock moves there
- runrun 			  -> copy of SKWTB_main for multiple runs
- SFM2 				  -> Social Force Model implementation
- SKWTB_main 			  -> main 
- TestEstimateComparison 	  -> compare test files
- TestEstimatePosition 		  -> test EstimatePosition.m
- TestEstimatePositionNoCons 	  -> test EstimatePositionNoCons.m 
- TestOccupancyMap 		  -> build occupancy map (NOT A TEST! 
				     Too dangerous changing now the name of the function!)
